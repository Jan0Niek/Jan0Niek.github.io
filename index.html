Wat mij betreft

Mijn naam is Yannick (Eleveld), ik ben een {huidige leeftijd}-jarige die de Coderclass op het Metis Montessori Lyceum heeft gevolgd.

Door de aan de coderclass mee te doen, is het mogelijk allerlei coole dingen te maken, zoals mijn altijd actuele leeftijd hierboven. Als je over een jaartje weer op deze pagina belandt, dan staat er een hoger getalletje! Stijgende getalletjes zijn altijd leuk.

Ook organiseert de Coderclass veel gave activiteiten, zo was er ook een reis naar San Francisco, om tech-bedrijven in Silicon Valley te bezoeken. Ook hebben mijn vrienden en ik in San Francisco deze foto gemaakt nadat we over de hele Golden Gate Bridge hadden gelopen. Verder organiseert de Coderclass ook jaarlijks game-jams, je zal nog wel wat afgeraffelde game-jam-games tegenkomen hier.

Het is erg leuk, maar vooral ook superleerzaam! Ik heb er veel geleerd over van alles en nog wat.

Opgedane ervaring

Vanuit Coderclass kon ik stapsgewijs dus over van alles en nog wat leren. Ik heb mijn kennis en ervaring met behulp van Scratch, Python turtle & Blender-kubusbomen uitgewerkt tot steeds hogere niveaus, tot aan mijn niveau van nu. Natuurlijk is het niet bij kubusbomen en Scratch gebleven. Ondertussen ben ik niet alleen vloeiend in Nederlands en Engels, maar beheers ik ook Python, JavaScript, TypeScript, C++, C# en Java allemaal best behoorlijk. De meeste van deze talen zijn mij via de Coderclass geleerd, maar door mijn deelname aan de Coderclass ging ik ook vaker zelf programmeerprojectjes starten. Ook kan ik prima omgaan met Unity, p5js, q5js en p5play (en Blender), toch is mijn favoriete game engine nu Godot. Ik kan wel beschrijven wat ik in elke taal kan doen, maar dat kan ik beter laten zien door middel van wat er overgebleven is van mijn projecten. Helaas ben ik best wel eens ergens aan begonnen zonder het af te maken, helaas ben ik heel wat Blender-renders kwijt, toch is er wel wat overgebleven qua projecten.

Niet te vergeten: ik gebruik trouwens Arch-Linux. Althans, CachyOS, wat gebasseerd is op Arch. Ik vind zelf dat ik prima met Linux (uiteraard ook nog wel met Windows) overweg kan. Ook gebruik ik voor deze pagina Git als versiebeheersysteem, Github als cloud-back-up en als webhosting.
